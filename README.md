# Banking Operations Analysis

Приложение для анализа банковских операций из Excel-файла.

Проект разработан в рамках учебной курсовой работы и позволяет анализировать банковские операции, формировать данные для веб-страниц, выполнять пользовательские сервисы и создавать отчеты.

## Реализованные задачи

### Веб-страницы

#### Главная

Формирует данные для главной страницы приложения:

- приветствие пользователя;
- информация по банковским картам;
- топ-5 транзакций по сумме платежа;
- курсы валют;
- стоимость акций.

### Сервисы

#### Выгодные категории повышенного кешбэка

Сервис анализирует операции пользователя и определяет категории, по которым можно получить наибольший кешбэк за выбранный месяц.

### Отчеты

#### Траты по категории

Формирует отчет по расходам за выбранную категорию за последние три месяца относительно указанной даты.

Результат отчета автоматически сохраняется в файл с помощью декоратора.

## Используемые технологии

* Python 3.14

* Poetry

## Особенности работы

- Курсы валют запрашиваются через внешний API.
- Стоимость акций запрашивается через внешний API.
- Результат зависит от доступности сервисов и действующих лимитов запросов.

## Установка проекта

Клонировать репозиторий:

```bash
git clone <https://github.com/Jd4rc/lsn_banking_operations_analysis.git>
```

Перейти в каталог проекта:

```bash
cd lsn_banking_operations_analysis
```

Установить зависимости:

```bash
poetry install
```

Активировать виртуальное окружение:

```bash
poetry shell
```

## Настройка переменных окружения

Создать файл `.env` на основе шаблона:

```bash
cp .env_template .env
```

Указать необходимые API-ключи:

```env
EXCHANGE_RATES_API_KEY=your_exchange_api_key
ALPHA_VANTAGE_API_KEY=your_alpha_vantage_api_key
```

## Запуск тестов

Запуск всех тестов:

```bash
pytest
```

Проверка покрытия:

```bash
pytest --cov=src
```
